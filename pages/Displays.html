Displays

Setting the Item Display Leaderboard

1Go to ADMIN > Displays > Bid Prices Display Options

2Review all fields and make any changes needed. *Default settings are usually best.

3Either copy the link from the box or click the View Bid Prices Display button.

4Double Check that all fields are fulfilled properly and send the link to the client.

Setting the Donation Display

1Go to ADMIN > Displays > Donation Thermometer Display Options

2Set colors to match the client's preferences.

3In the Show Names field, be sure to choose No if the client does not want donor names shown.

4In the Show Donation Amounts field, change it to No if the client does not want the amount shown.

3Either copy the link from the box or click the View Donations Display button.

4Double Check that all fields are fulfilled properly and send the link to the client.
